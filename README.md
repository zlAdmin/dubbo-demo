# dubbo-demo
dubbo测试项目

#springboot+dubbo
    基于springboot+dubbo的模板项目
    
   项目启动流程：先install dubbo-demo-api项目
                 运行dubbo-demo-provider的启项目
                 运行dubbo-demo-consumer的启项目
                 
   进入浏览器输入：http://localhost:8090/sayHello?name=word
   进入浏览器输入：http://localhost:8091/actuator/health    检查状态
